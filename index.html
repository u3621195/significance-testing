How It Works
Methodology, thresholds, and interpretation
📐
Z-test formula
z = (p₁ − p₂) ÷ √[(p₁(1−p₁)/n₁) + (p₂(1−p₂)/n₂)]
where p is the proportion and n is the base size.
📊
Three-tier notation
B = sig. at 99% (z ≥ 2.576)
B = sig. at 95% (z ≥ 1.960)
b = sig. at 90% only (z ≥ 1.645)
💼
Research convention
Follows standard market research cross-tab reporting. Compatible with SPSS-style notation. Each cell shows % value and letters of columns it significantly exceeds.
🎚️
Flexible confidence levels
Select one level (e.g. 95% only), two levels (e.g. 95%+90%), or all three. The summary table uses the letter hierarchy to show the strongest level of significance.
⚠️
Validity requirements
Base size (n) must be a positive number. Percentage must be between 0 and 100. Invalid rows are excluded from comparisons and shown as "Invalid" in the matrix.
🔍
Reading the matrix
The pairwise matrix shows the exact z-score for each pair. Green = higher, red = lower, relative to the selected confidence threshold(s). Diagonal = self-comparison.
